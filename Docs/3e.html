Editing Stations
Cargo Ratings
The checkmark at each cargo is intended for use with the "selectgoods" option in TTDPatch, removing it will preform a reset of just that type of cargo in this station, unlike the "cht:resetstation" in TTDPatch that'll reset all cargo everywhere...


Add Buildings
Warning: This option have only been tested in tempare climate, and I think it'll preform badly elsewhere.

This will fill all unused squares in the catchment area with buildings, make sure you reserve any areas you'll need in the future before using this. In case you wish the city to look natural you'll also need to make some roads in the area prior to using the option.

Note: The editor map is NOT updated after using this option, so the new buildings ain't visible until the map is refreshed, you can do this by reselecting the mapsize in the options menu.

Note: To insure everything in the game is updated proberly the buildings are placed "under construction", once the game finishes the buildings, the statistics and cargo are updated.


Add Rem(ote) Part
This option will place a bus or truck stop at a location away from the station, this can ofcourse be used instead of "walking" a station, but the new location don't have to be at the same height as the rest of the station. Unfortunally this requires a bit preparations first.

1) Make sure the station havn't got a bus or truck station, if it have neither then you can make 2 remote parts in one go. Note that the busstop will be placed before the truckstop !!

2) Pick the spot for the remote part, it must be within the catchment area of a part of the station. (That's max 4 squares away). The function is a rather low-level one for the time be, so the entrence to the new part is always to the NE. Build this on the square:


(All the pieces of plain railroad you can build)

3) Now load the game into this editor, select the station and hit the button...

Hint: It's perfectly legal to use the catchment area of one remote part to set the next.
