So What's This Then ??
This is the home of the TTDX Editor, a support utility for Transport Tycoon Deluxe (Windows 95 and Dos versions). I don't know yet what the program will be capable of doing later on, but so far it's starting out as a save-game editor.

This site is a member of WebRing.
To browse visit Here.

Requirements:
Windows 98/ME/NT4/2K/XP
Internet Explorer 6 OR Windows Scripting 5.6
Visual Basic 6 Sp5 runtimes
A Desktop of 800x600 is recomended
A Desktop of 1024x768 or better if using the larger maps

Known Problems:
Run-time error '430', Class does not support Automation...
Reason: The scripting-runtime library is to old on your system, the file is normally found in the "windows/system" or "windows/system32" folders and is called "scrrun.dll", the version used for building the editor is 5.6.0.6626. This version is included with Internet Explorer 6, but can also be downloaded as a part of the windows script 5.6 pack. (Link above)
