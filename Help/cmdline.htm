Command Line Options

Command line options are not often seen in Windows programs, yet many programs do actually have them. TTDX Editor is no different. The parameters the Editor accepts are shown below:

  • TTDXEdit.exe %filepath%
    Loads the file directly into the editor. Both normal TTD files and uncompressed hdr/dta files may be used.
  • TTDXEdit.exe %filepath% /S
    Loads the file and saves it again in normal savegame format. This can be used to fix games saved in the Windows version of TTD that won't load into the DOS version. The editor will not appear on screen.
  • TTDXEdit.exe %filepath% /SU
    Converts the file to uncompressed format.
