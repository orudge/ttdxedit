File Associations

Associating files to certain programs allows for direct access in Explorer. If you select a movie clip, for example, it is loaded into a program capable of playing it. Transport Tycoon saved games have no association by default - they are unknown to Windows. However this editor can assign itself to the saved games so they can be loaded by simply double-clicking them in Explorer.

You can associate TT saved games with files by selecting the File Assocations item from the Options menu. You can select which of TTD's filetypes you want to associate. If another program (such as Transport Tycoon Saved Game Manager) has associated itself with some or all of the filetypes, you can choose to overwrite the settings or simply 'blend in' with them. After setting this up you'll find some new options in the right-click pop-up menu for these types of files:

  • Edit with TTDX Editor
    Opens the selected game for editing with TTDX Editor.
  • Repack to Game Format
    Decompresses and then recompresses the selected game. This fixes the problem when loading saved games from the Windows version of TTD into the DOS version.
  • Make Unpacked File
    Decompresses the selected game into its unpacked form. This will create a .s??dta and .s??hdr file for the game (where .s?? is the current extension - eg, .sv1).

Notes

Selecting the "plug-in" option does NOT change the default action of the filetype(s). It simply places the TTDX Editor options into the menu.

There is no option to remove the associations yet. This will be added in a future version of TTDX Editor.
